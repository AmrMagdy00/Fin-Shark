# FinShark Documentation Index

الهدف من فولدر `docs/` هو شرح المشروع بالكامل بشكل عملي: **إزاي تشغّله، إيه مكوناته، إزاي الـ request بيمشي جوه السيستم، وإيه شكل الداتا والـ API**.

## محتويات التوثيق (اقرأ بالترتيب)

- `01-QUICKSTART.md`: تشغيل المشروع + إعداد الداتابيز
- `02-ARCHITECTURE.md`: المعمارية (Controllers/Repositories/Services/DTOs/Mappers)
- `03-CONFIGURATION.md`: مفاتيح الإعدادات في `appsettings.json` ومعانيها
- `04-AUTHENTICATION_JWT.md`: Identity + JWT + Roles + Swagger auth
- `05-DATABASE_SCHEMA.md`: الكيانات والجداول والعلاقات (حسب الـ migration)
- `06-API_ENDPOINTS.md`: جميع الـ endpoints مع الـ routes والـ verbs والـ query params
- `07-FOLDERS_AND_FILES.md`: شرح كل فولدر وملف مهم
- `08-NOTES_AND_KNOWN_ISSUES.md`: ملاحظات تقنية مهمة/نِقَاط تحتاج انتباه
